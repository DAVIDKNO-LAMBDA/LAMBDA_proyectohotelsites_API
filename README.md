# LAMBDA_proyectohotelsites_API
LAMBDA Sites Hotels - Proyecto Web Descripción Este proyecto es una migración de dashboards y KPIs desde Power BI a una aplicación web moderna, utilizando Django para el backend y React + TypeScript para el frontend. Permite la gestión de usuarios con roles, autenticación JWT, visualización de métricas y KPIs, y filtrado dinámico de datos.
